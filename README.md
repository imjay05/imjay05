### 🚀 Featured Projects

#### 🩺 [HealthLense](https://github.com/imjay05/healthlense) — AI-Powered Medical Report Analysis Platform
Turns confusing medical reports into plain-language insights in English, Hindi, or Marathi, with a real-time nearby-labs finder.
- Cost-aware AI pipeline with SHA-256 file dedupe before any Cloudinary/Groq call
- Multi-modal document pipeline (PDF/DOCX/images) into a single vision-model analysis path
- Dependency-free MongoDB TTL rate limiter with a fail-open policy
- Real-time lab search over an authenticated Socket.IO channel
- `React` `Node.js` `Express` `MongoDB` `Socket.io` `Groq` `Cloudinary`

#### 📈 [ArcTrade](https://github.com/imjay05/arctrade) — Full-Stack Stock Trading Platform
A MERN trading simulator with transactional order execution and a live NSE market data pipeline.
- ACID-safe order placement via MongoDB multi-document transactions (wallet, orders, holdings)
- Real weighted-average cost basis calculation for holdings
- Razorpay payment integration with server-side HMAC-SHA256 signature verification
- Hand-built HTML5 Canvas candlestick chart, no charting library
- `React` `Node.js` `Express` `MongoDB` `Razorpay` `JWT`

#### 💬 [Vibley](https://github.com/imjay05/vibley) — Real-Time Messaging Platform
A friends-only chat app built around presence, not permanence — messages have a rolling 7-day window and moods expire in 24 hours.
- Optimistic send with clean client-side rollback on failure
- Delivery status that survives disconnects and catches up retroactively
- MongoDB aggregation pipelines for unread counts and conversation ordering
- httpOnly, sameSite-aware auth cookies — JWT never touches localStorage
- `React` `Node.js` `Express` `MongoDB` `Socket.io` `Cloudinary` `Tailwind`
